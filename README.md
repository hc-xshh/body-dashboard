# Body Dashboard

Daemon 的个人身体数据仪表盘。自动解析体测截图，结合最新体测、当日周计划与历史趋势，生成一个适合每天查看的 **状态 → 执行 → 趋势** 页面。

**正式线上地址**：`https://body-dashboard.pages.dev`

## 当前功能

### 1. 顶部摘要区
- 展示最新体测日期 / 时间
- 展示体型、身体年龄、当天训练主题
- 展示**体测同步状态双态提示**：
  - 已同步：当天建议基于今日最新体测
  - 未同步：当天建议沿用最近一次体测，并明确提示日期

### 2. 阅读路径导航
- 提供可点击的阅读路径：**状态 → 执行 → 趋势**
- 适合每天快速打开后按固定顺序浏览

### 3. 状态区
- 🃏 核心指标卡片：体重、体脂率、BMI、基础代谢、内脏脂肪、水分、骨量、综合得分
- 🕸️ 身体成分雷达图
- 💡 基于阈值规则的健康建议

### 4. 执行区
- 🏋️ 今日训练执行单
- 🧴 今日护肤流程
- 🥗 今日推荐饮食

### 5. 趋势区
- 📊 体重 & 体脂率趋势折线图
- 💪 肌肉 / 水分 / 去脂体重趋势
- 📋 历史记录表格

## 数据来源与生成逻辑

- 体测数据存储在 `src/data/measurements.json`
- 由 Hermes Agent 解析体测截图后更新数据
- 当日训练 / 护肤 / 饮食内容来自本地计划配置
- 健康建议由阈值规则生成
- 如果当天没有新体测，页面仍可用，但会明确提示当前建议使用的是最近一次体测数据

>
> Vercel 已移除，不再作为部署平台或访问入口。
